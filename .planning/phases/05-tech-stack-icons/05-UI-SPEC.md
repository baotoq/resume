## Spacing Scale

Declared values (multiples of 4, matching project's Tailwind v4 rhythm):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-icon gap within the tech stack row (`gap-1`) |
| sm | 8px | Gap between icons in the row (`gap-2`) |
| md | 16px | Default element spacing |
| lg | 24px | Card padding horizontal/vertical (`px-6 py-6`) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

**Icon row spacing (prescriptive):**
- Top margin from header block to icon row: `mt-3` (12px)
- Bottom margin from icon row to bullets `<ul>`: `mb-3` (12px) — the bullets `<ul>` already carries `mt-4`; set `mb-3` on the icon row so combined visual gap is ~16px
- Gap between icons within the row: `gap-2` (8px)

Exceptions: 44px touch target not applicable — icons are decorative display elements, not interactive.

Source: 05-CONTEXT.md Claude's Discretion, existing `WorkExperience.tsx` internal rhythm (`mt-4`, `gap-2`, `gap-3`, `gap-6`)

## Component Contract

### `TechStackIcons.tsx`

**Type:** Server Component (no browser APIs, no state, no event handlers)

**Props:**
```ts
interface TechStackIconsProps {
  stack?: string[]
}
```

**Render contract:**
- `stack` is `undefined` or empty array → render `null` (no DOM node, no empty space)
- `stack` has entries → render a `<div>` with `flex flex-wrap items-center gap-2 mt-3 mb-3`
- For each tech name in `stack`:
  - Look up the name in the slug map (case-insensitive)
  - **Found in map:** render `<i className="devicon-{slug}-{variant} colored text-2xl" title="{tech name}" aria-label="{tech name}" />`
  - **Not in map:** render `<span className="bg-zinc-100 text-zinc-600 rounded-full px-2 py-0.5 text-xs">{tech name}</span>`

**Icon element accessibility:** Each `<i>` carries `title="{tech name}"` and `aria-label="{tech name}"`. The wrapping row carries `aria-label="Tech stack"`.

## Interaction States

| State | Visual |
|-------|--------|
| Default (icons loaded) | Colored Devicons glyphs, 24px, `gap-2` row |
| Icon load failure (CDN down) | Devicon `<i>` renders as blank space — acceptable degradation; no fallback needed since CDN failure is transient |
| Unknown tech name | zinc-100 / zinc-600 rounded-full text pill at 12px |
| No tech_stack field | Component renders null — zero layout impact |

No hover states. No click states. Icons are decorative display elements only.
