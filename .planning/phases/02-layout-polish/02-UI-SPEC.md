## Animation Contract

This section is new to Phase 2 (LAYOUT-02).

### Entry Animation

| Property | Value |
|----------|-------|
| Initial state | `opacity: 0, y: 16` |
| Animate state | `opacity: 1, y: 0` |
| Transition duration | `0.4s` |
| Transition easing | `easeOut` |
| Trigger | `whileInView` |
| Repeat | Once only — `viewport={{ once: true }}` |

Source: CONTEXT.md D-02, D-04.

### Stagger

| Property | Value |
|----------|-------|
| Delay between sections | `0.1s` (Header: 0s, WorkExperience: 0.1s, Skills: 0.2s) |
| Stagger direction | Top → bottom (Header → WorkExperience → Skills) |
| Implementation | `transition={{ delay: index * 0.1 }}` on each `AnimateIn` wrapper, or Framer Motion `staggerChildren: 0.1` on parent `motion.div` |

Source: CONTEXT.md D-03 (stagger pattern); delay value 0.1s chosen at Claude's discretion (within suggested 0.1s–0.15s range).

### AnimateIn Wrapper

| Property | Value |
|----------|-------|
| Component name | `AnimateIn` |
| File path | `src/components/AnimateIn.tsx` |
| Directive | `'use client'` — required by Framer Motion |
| Props | `children: React.ReactNode`, `delay?: number` (default `0`) |
| Element rendered | `motion.div` wrapping children |
| Usage | Wrap each section card in `page.tsx` — do NOT add `'use client'` to Header, WorkExperience, or Skills |

Source: CONTEXT.md specifics section (Server Component constraint).

### Reduced Motion

| Property | Value |
|----------|-------|
| Handling | Framer Motion respects `prefers-reduced-motion` by default via `useReducedMotion`. No additional code needed. |

## Accessibility Contract

Inherited from Phase 1, plus animation additions:

- `prefers-reduced-motion: reduce` — Framer Motion's default behavior disables motion when user has reduced motion enabled. No additional implementation needed.
- Animated wrappers (`AnimateIn`) use `motion.div` — semantically neutral, does not alter heading hierarchy or landmark structure.
- All Phase 1 accessibility rules remain in effect: heading hierarchy (`h1` → `h2` → `h3`), contrast ratios, anchor descriptive text.
