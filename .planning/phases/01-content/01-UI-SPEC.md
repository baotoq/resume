## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Bullet gap, inline label gap |
| sm | 8px | Contact link spacing, skill tag gaps |
| md | 16px | Card inner padding (horizontal), list item gap |
| lg | 24px | Card inner padding (vertical), between card header and body |
| xl | 32px | Between section cards |
| 2xl | 48px | Top/bottom page padding |
| 3xl | 64px | Max-width content container horizontal margin |

Exceptions: none

Source: default 8-point scale; usage mapped to Phase 1 elements (header card, experience cards, skills card).

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.1 |

Details:
- Display (28px / 600): name in header
- Heading (20px / 600): job title in header; section titles (Work Experience, Skills)
- Body (16px / 400): experience bullet points, skills values, contact link text
- Label (14px / 400): company name, role, date range, skills category labels (e.g. "Languages"), muted metadata

Font stack: `font-sans` → `var(--font-geist-sans)`, Geist Sans, system-ui, sans-serif.

Source: CONTEXT.md D-02 (zinc-900 header text, zinc-700 body text); default professional typography scale applied.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#fafafa` (zinc-50) | Page background |
| Secondary (30%) | `#ffffff` (white) | Section cards (Header, Work Experience, Skills) |
| Accent (10%) | `#4f46e5` (indigo-600) | Email link, GitHub link, LinkedIn link — interactive anchors only |
| Muted | `#a1a1aa` (zinc-400) | Section header labels, date ranges, category labels |
| Text primary | `#18181b` (zinc-900) | Name, section headings |
| Text body | `#3f3f46` (zinc-700) | Bullet points, body copy |
| Destructive | not applicable | No destructive actions in Phase 1 |

Accent reserved for: email anchor, GitHub link anchor, LinkedIn link anchor — no other elements use indigo-600.

Source: CONTEXT.md D-02 explicitly names zinc-50 background, white cards, zinc-900 header text, zinc-700 body text, indigo-600 accent, zinc-400 muted.

## Interaction States

Phase 1 has minimal interaction (content rendering only):

---

## Accessibility Contract

- All anchor elements have descriptive text (no bare URLs)
- GitHub link aria-label: "GitHub profile" (if icon-only, but here it is text)
- LinkedIn link aria-label: "LinkedIn profile" (if icon-only, but here it is text)
- Heading hierarchy: `h1` for name, `h2` for section titles (Work Experience, Skills), `h3` for company/role
- Color contrast: zinc-900 on white = 19:1 (exceeds 4.5:1 WCAG AA); zinc-700 on white = 9.4:1; indigo-600 on white = 4.6:1 (passes AA)
- `lang="en"` already set on `<html>` in layout.tsx
