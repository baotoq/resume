Class explanation (UI-SPEC confirmed):
- `mt-4` — 16px top margin separating bio from the contacts row (matches existing Header vertical rhythm: mt-1 name→title, mt-4 title→contacts, mt-4 contacts→bio)
- `text-base` — 16px body size
- `leading-relaxed` — 1.625 line height for prose readability
- `text-zinc-600` — #52525b, one step lighter than bullet text (zinc-700), signals supporting context

Guard requirement (D-02 / Pitfall 2 from RESEARCH.md): The `{resume.bio && ...}` guard is MANDATORY. Without it, an empty `<p>` with mt-4 margin renders when bio is undefined, creating invisible whitespace.
