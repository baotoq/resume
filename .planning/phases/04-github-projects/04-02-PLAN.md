Create `src/components/resume/Projects.tsx` with named export `ProjectsSection`.

**Imports:**
- `Section` from `./Section`
- `type GitHubRepo` from `@/types/resume`
- `languageColors` from `@/data/githubColors`
- `GithubOutlined`, `StarOutlined`, `ForkOutlined` from `@ant-design/icons`

**Props interface:**
```typescript
interface ProjectsProps {
  repos: GitHubRepo[];
}
```

**Component logic:**
- If `repos.length === 0`, return `null` (hide section entirely when empty)
- Wrap the entire output in `<div className="print:hidden">` to exclude from print/PDF
- Inside that, use `<Section title="Projects" icon={<GithubOutlined />}>`
- Render a responsive grid: `grid grid-cols-1 md:grid-cols-2 gap-5`

**Each card is an `<a>` tag** (entire card is clickable):
- `href={repo.url}` `target="_blank"` `rel="noopener noreferrer"`
- Card classes: `block p-5 rounded-2xl bg-[var(--muted)] border border-[var(--border)] hover:shadow-md hover:border-[var(--accent)]/50 transition-all duration-200`
- **Repo name:** `text-base font-semibold text-[var(--accent)]` â€” shows the repo name
- **Description:** `text-sm text-[var(--secondary)] mt-1.5 line-clamp-2` â€” clamp to 2 lines
- **Stats row** at bottom with `flex items-center gap-4 mt-3 text-xs text-[var(--secondary)]`:
  - **Language** (if not null): colored dot (`w-2.5 h-2.5 rounded-full inline-block`) using `languageColors[repo.language]` with a fallback gray `#8b8b8b` if language not in map, then language name text
  - **Stars:** `<StarOutlined />` icon + `repo.stars` (use `Intl.NumberFormat("en", { notation: "compact" })` if > 999)
  - **Forks:** `<ForkOutlined />` icon + `repo.forks`
  - **Updated:** Format `repo.updatedAt` using `new Date(repo.updatedAt).toLocaleDateString("en-US", { month: "short", year: "numeric" })` â€” shows like "Mar 2026"

Follow project conventions: named export only, no default export, `interface ProjectsProps` (no I prefix), destructured props, Tailwind class order: layout > spacing > sizing > colors > typography > effects > states.
