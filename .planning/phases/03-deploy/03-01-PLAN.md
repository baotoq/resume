  <action>
Create `.github/workflows/deploy.yml` with a two-job workflow (build + deploy) per D-01 and D-04.

The workflow file must contain exactly this content:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: pages
  cancel-in-progress: false

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: npm

      - name: Install dependencies
        run: npm ci

      - name: Build
        run: npm run build

      - name: Configure Pages
        uses: actions/configure-pages@v5

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: out

  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    needs: build
    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

Key details per D-04:
- Trigger: `push` to `main` branch (also `workflow_dispatch` for manual runs)
- Build: `npm ci && npm run build` (two steps for clarity in logs)
- Artifact: `out/` directory via `actions/upload-pages-artifact@v3`
- Deploy: `actions/deploy-pages@v4`
- Uses `actions/configure-pages@v5` per D-04

Security:
- `permissions` block follows principle of least privilege: only `contents: read`, `pages: write`, `id-token: write`
- `concurrency` prevents parallel deploys that could conflict
- `cancel-in-progress: false` ensures a deploy in progress finishes (no partial deploys)

The `environment.url` uses `${{ steps.deployment.outputs.page_url }}` — this is the standard GitHub Actions expression that resolves to the deployed Pages URL after the `actions/deploy-pages` step completes.

Note: The directory `.github/workflows/` does not exist yet -- create both directories.
