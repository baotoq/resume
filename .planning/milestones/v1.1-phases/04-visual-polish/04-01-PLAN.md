Replace the inner `<div className="flex flex-col gap-6">` and its contents with a timeline rail wrapper. The full structure:

```tsx
{/* Rail wrapper — relative context for line + dots */}
<div className="relative pl-5 sm:pl-7 flex flex-col gap-6">

  {experience.map((entry, index) => {
    const isCurrent = entry.endDate === null
    const isLast = index === experience.length - 1

    return (
      <div key={index} className="relative">
        {/* Vertical line segment — extends from dot to next entry; omit on last entry (TIMELINE-04 per D-10) */}
        {!isLast && (
          <div
            className="absolute -left-[19px] sm:-left-[23px] top-[28px] bottom-[-24px] w-0.5 bg-zinc-200"
            aria-hidden="true"
          />
        )}

        {/* Timeline dot (TIMELINE-02 per D-09) */}
        <div
          className={`absolute -left-[22px] sm:-left-[26px] top-[22px] w-3 h-3 rounded-full ${
            isCurrent
              ? "bg-indigo-600"
              : "border-2 border-zinc-300 bg-white"
          }`}
          aria-hidden="true"
        />

        {/* Card */}
        <article className="rounded-xl border border-zinc-200 bg-white px-6 py-6 shadow-sm">
          {/* Header — restructured with LogoImage (per D-06) */}
          <div className="flex items-start gap-3">
            <LogoImage src={entry.logo_url} alt={`${entry.company} logo`} />
            <div className="flex flex-col flex-1 gap-1 sm:flex-row sm:items-baseline sm:justify-between">
              <div>
                <h3 className="text-sm font-normal text-zinc-700">{entry.company}</h3>
                <p className="text-sm font-normal text-zinc-700">{entry.role}</p>
              </div>
              <span className="text-sm font-normal text-zinc-400">
                {formatDateRange(entry.startDate, entry.endDate)}
              </span>
            </div>
          </div>
          {/* Bullets — unchanged */}
          <ul className="mt-4 flex flex-col gap-2">
            {entry.bullets.map((bullet, i) => (
              <li
                key={i}
                className="text-base leading-relaxed text-zinc-700 pl-4 relative before:content-[''] before:absolute before:left-0 before:top-[10px] before:h-1.5 before:w-1.5 before:rounded-full before:bg-zinc-300"
              >
                {bullet}
              </li>
            ))}
          </ul>
        </article>
      </div>
    )
  })}
</div>
```

Key implementation notes:
- `pl-5 sm:pl-7` — mobile rail 20px, desktop 28px (per D-11)
- Line segments use per-entry approach with `!isLast` guard (TIMELINE-04 per D-10)
- `bottom-[-24px]` bridges the `gap-6` (24px) between cards
- `top-[28px]` for line start = card top padding (24px) + ~4px to reach dot center
- `top-[22px]` for dot = card top padding (24px) - 2px visual offset for 12px dot centering
- Dot left offsets: `-left-[22px] sm:-left-[26px]` to center the 12px dot on the rail
- Line left offsets: `-left-[19px] sm:-left-[23px]` to center the 2px line (w-0.5) on the same axis
- These pixel values are estimates — visual tuning may be needed in dev (flagged in RESEARCH.md)
- `<h2>` heading stays outside the rail wrapper — line does not extend through it
