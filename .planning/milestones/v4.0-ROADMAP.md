## Key Decisions

- **globals.css merge manual** — `shadcn@latest init` destructively overwrites; Geist font vars and `@theme inline` block must be preserved by hand.
- **All token hex values converted to oklch** during merge — no mixed hex/oklch token system.
- **Badge applies only to unknown-tech fallback pill** — NOT to recognized SVG icon entries.
- **Separator must NOT replace inline `·` contact-row dots** — block element would break flex layout.
- **Plain `npm install`** — no `--legacy-peer-deps` needed; all packages declare React 19 peer deps.
- **Phase 14 used zero className overrides** — D-03/04/05 shadcn defaults trusted.
