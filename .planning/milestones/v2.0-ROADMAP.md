## Overview

Migrated deployment from GitHub Pages static export to Vercel, unlocking full Next.js 16 runtime capabilities. Removed all static-export constraints, configured Vercel native Git integration, added HTTP security headers, converted logos to next/image with Clearbit remotePatterns, made env vars server-only, and decommissioned GitHub Pages.

## Milestone Summary

**Key Decisions:**

- Vercel native Git integration chosen over GitHub Actions + Vercel CLI — fewer moving parts, no secrets management
- All static-export constraints removed atomically in a single plan (not incrementally)
- Quick wins (IMG-01, SEC-01, CFG-01) bundled into Phase 7 alongside the migration
- VERCEL-04 (decommission Pages) isolated to Phase 8 — irreversible step, must verify Vercel healthy first
- WR-01: guard readFileSync with try/catch — throw Error for App Router to catch
- WR-02: LogoImage props corrected from ButtonHTMLAttributes to HTMLAttributes<HTMLDivElement>
