## Component Layout Contract

### Card: `<article>` element

```
rounded-xl border border-zinc-200 bg-white p-6 shadow-sm
```

Internal structure — two-column responsive header (mobile stacks, sm+ aligns row):

```
<div className="flex flex-col sm:flex-row sm:items-start sm:justify-between gap-1 sm:gap-0">
  <!-- LEFT COLUMN -->
  <div className="flex flex-col gap-1">
    <h3>  degree title    text-lg font-semibold text-zinc-900
    <p>   institution     text-base text-zinc-700
  </div>

  <!-- RIGHT COLUMN -->
  <span>  date range     text-sm font-semibold text-zinc-500 sm:text-right
```

Optional details paragraph (rendered only when `details` field is non-empty):

```
<p className="mt-4 text-base leading-relaxed text-zinc-700">
  {entry.details}
</p>
```

### Section wrapper

```
<section>
  <h2 className="text-xl font-semibold leading-[1.2] text-zinc-900 mb-6">
    Education
  </h2>
  <div className="flex flex-col gap-6">
    {/* cards */}
  </div>
</section>
```

No timeline rail, no dot indicators — confirmed out of scope (REQUIREMENTS.md, CONTEXT.md D-01).

### Responsive breakpoint

- Mobile (`< sm`): degree+institution stacked left, date range below on same left column
- `sm` and above: degree+institution left, date range right — `sm:flex-row sm:items-start sm:justify-between`

Pattern mirrors `WorkExperience.tsx` lines 59–79 exactly.

## Animation Contract

Component is NOT self-animated. `page.tsx` wraps it externally:

```tsx
<AnimateIn delay={0.2}>
  <EducationSection education={resume.education ?? []} />
</AnimateIn>
```

- Delay: `0.2` — one step after WorkExperience (`delay={0.1}`) in stagger sequence
- Animation: `opacity: 0 → 1`, `y: 16 → 0`, `duration: 0.4`, `ease: easeOut`, `once: true`
- Source: `AnimateIn.tsx`, `page.tsx` stagger pattern (Header=0, WorkExperience=0.1)

## Data Contract

`EducationEntry` interface (after Phase 11 type update):

```ts
export interface EducationEntry {
  degree: string;
  institution: string;
  startDate: string;   // "YYYY-MM" format
  endDate: string | null;
  logo_url?: string;   // ignored — no logo rendered
  link?: string;       // ignored — no link rendered
  details?: string;    // new optional field — plain prose paragraph
}
```

Date formatting: reuse `formatDateRange()` from `WorkExperience.tsx` (same function signature, same output: "Sep 2014 – Jun 2018").

Current live data (Ton Duc Thang University): `startDate: "2014-09"`, `endDate: "2018-06"` → renders as "Sep 2014 – Jun 2018".
