   ```bash

   ```

2. Create `scripts/generate-og-image.ts` — a one-time script (NOT a build step) that generates the OG image:
   - Use satori to render a React-like JSX structure to SVG, then resvg to convert SVG to PNG
   - Image dimensions: 1200 x 630 px
   - Design layout:
     - Background: warm beige gradient (matching site theme, e.g., #faf8f5 to #f5f0eb)
     - Left side: Name "To Quoc Bao" in large bold text (dark warm gray), "Senior Software Engineer" subtitle in teal (#0d9488)
     - Bottom area: Key skill tags (C#, TypeScript, Golang, Kubernetes, AWS, Azure) as small rounded pills
     - Subtle decorative element: teal accent line or shape
     - Clean, professional, minimal — matching the resume site aesthetic
   - Font: Use a system sans-serif or download Plus Jakarta Sans font file for satori (satori requires font data as ArrayBuffer)
   - Output to `src/app/opengraph-image.png`

3. Run the script once with tsx:
   ```bash
   npx tsx scripts/generate-og-image.ts
   ```
