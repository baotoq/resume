1. Install tsx as devDependency: `npm install -D tsx`

2. Create `scripts/fetch-github.ts`:

**Config object at top of file:**
```typescript
const CONFIG = {
  username: "baotoq",
  excludeForks: true,
  excludeArchived: true,
  minStars: 0,
  maxRepos: 6,
  excludeRepos: ["baotoq"], // GitHub profile repo
};
```

**GitHubApiRepo interface** (internal to script, not exported — this is the raw API shape):
```typescript
interface GitHubApiRepo {
  name: string;
  html_url: string;
  description: string | null;
  language: string | null;
  stargazers_count: number;
  forks_count: number;
  pushed_at: string;
  fork: boolean;
  archived: boolean;
  topics: string[];
}
```

**fetchRepos function:** Calls `https://api.github.com/users/${CONFIG.username}/repos` with query params `sort=pushed&direction=desc&per_page=100&type=owner`. Set header `Accept: application/vnd.github.v3+json`. If `GITHUB_TOKEN` env var exists, add `Authorization: Bearer ${token}` header for higher rate limits.

**filterRepos function:** Takes `GitHubApiRepo[]`, applies CONFIG filters (excludeForks, excludeArchived, minStars, excludeRepos), then slices to maxRepos.

**transformRepo function:** Maps `GitHubApiRepo` to the app's `GitHubRepo` shape:
- `name` -> `name`
- `html_url` -> `url`
- `description ?? "No description"` -> `description` (handle null)
- `language` -> `language` (keep null)
- `stargazers_count` -> `stars`
- `forks_count` -> `forks`
- `pushed_at` -> `updatedAt`

**writeDataFile function:** Writes to `src/data/github.ts` using `fs.writeFileSync`. Output format:
```typescript
import type { GitHubRepo } from "@/types/resume";

// Auto-generated by scripts/fetch-github.ts on ${new Date().toISOString()}
// Do not edit manually — changes will be overwritten on next build
export const githubRepos: GitHubRepo[] = ${JSON.stringify(repos, null, 2)};
```

**Main execution:** Wrap in try/catch. On success, log count of repos written. On failure, log warning "GitHub API unavailable, using existing fallback data" and exit with code 0 (do NOT fail the build).

3. In `package.json`, add `"prebuild": "tsx scripts/fetch-github.ts"` to scripts. npm automatically runs prebuild before build. Do NOT modify the existing "build" script.
